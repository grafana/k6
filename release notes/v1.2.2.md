- [#5069](https://github.com/grafana/k6/pull/5069) Fixes group order in end of test summary when scenarios are used. 
- [#5070](https://github.com/grafana/k6/pull/5070) Adds nullish check to the new getByRole and add tests for other getBy* APIs nullish checks.
